# Python Automation Scripts

A curated collection of Python scripts designed to automate everyday system and data tasks. This repository is organized into five main categories for easy navigation and use:

1. Log Rotation & Backup – Automate log file management and system backups.
2. System Health Metrics – Monitor and report on key system performance indicators.
3. User Management – Simplify user account creation, updates, and maintenance.
4. Log Parsing – Efficiently analyze and extract insights from log files.
5. Report Generation – Generate structured reports from system or application data.
